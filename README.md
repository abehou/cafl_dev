# CAFL: Coding Agent For Law
This repo implements a coding agent framework specialized for legal tasks. The motivation for having this agent is that legal AI developers / researchers and practitioners who want AI in their workflows often work with scattered legal datasets and need to implement workflows to benchmark LLM/agent performance on these datasets of various formats. Much time is wasted in implementing pipelines to send data and process. CAFL is capable of flexibly dealing and processing datasets of various formats, and legal AI developers/researchers only need to specify the configuration of their dataset with natural language and can easily benchmark or have language models to work on their domain tasks.
